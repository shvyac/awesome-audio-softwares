## 音楽ビジュアライゼーション

- [projectM](https://github.com/projectM-visualizer/projectm) - MilkDropと互換性のある、クロスプラットフォームのオープンソース音楽ビジュアライゼーションライブラリ。OpenGLでリアルタイムにオーディオ反応型のビジュアルを描画する。
- [MilkDrop](https://en.wikipedia.org/wiki/MilkDrop) - Ryan Geissによって開発された、Winamp用の定番ハードウェアアクセラレーション音楽ビジュアライゼーションプラグイン。ビートに同期したユーザー作成「プリセット」で知られる。
- [Advanced Visualization Studio (AVS)](https://en.wikipedia.org/wiki/Advanced_Visualization_Studio) - NullsoftによるWinamp用の音楽ビジュアライゼーションプラグイン。ユーザーが独自のビジュアルエフェクトチェーンを構築できる。
- [G-Force](https://en.wikipedia.org/wiki/G-Force_(visualization_software)) - Andy O'Mearaによる人気の音楽ビジュアライゼーションプラグイン。精巧で有機的な3Dビジュアルが特徴。
- [harmony_visualizer](https://github.com/Frieve-A/harmony_visualizer) - Frieve-Aによる、MIDI入力のハーモニーを可視化するツール。

## オーディオエフェクト・処理

- [EffeTune](https://github.com/Frieve-A/effetune) - Frieve-Aによる、音楽鑑賞体験を向上させるためのリアルタイムオーディオエフェクトプロセッサー。

## オーディオ解析・ツール

- [Frieve's Sound Toolbox](https://github.com/Frieve-A/sound_toolbox) - Frieve-Aによる、ブラウザ上で動作するサウンド関連ツール集。プレビューオーディオプレーヤー、ABXテスター、デジタルサンプリングビジュアライザーなどを含む。
- [DIP Bench](https://github.com/Frieve-A/dipbench) - Frieve-Aによる、デジタルピアノのベンチマークツール。
- [Audio Review](https://github.com/Frieve-A/audioreview) - Frieve-Aによる、ヘッドフォン・スピーカー・アンプなどのオーディオ機器を客観的に評価するレビューサイトのソース。
